# Oscars Memory 2000

A classic Memory game developed in Python using the Tkinter GUI library. The objective is to find matching pairs of words hidden behind a grid of buttons. The game tracks the number of wrong attempts and features a persistent scoreboard system.

## Features

* **GUI Interface:** A 6x6 grid interface built with Tkinter.
* **Dynamic Word Loading:** The game reads word pairs from an external text file "memo.txt", allowing for easy customization of the game content.
* **Score Tracking:** Tracks the number of incorrect guesses in real-time.
* **High Score System:** Saves the player's name and score to a file "topplista.txt" upon winning. The scoreboard can be viewed via the options menu.
